Política de Privacidad - RPD Data

Última actualización: 13 de noviembre de 2024

1. Introducción

Esta Política de Privacidad describe cómo RPD Data ("la Aplicación") recopila, utiliza, almacena y protege tu información personal. Esta aplicación es una herramienta empresarial privada para la gestión de productos con escaneo de códigos de barras.

2. Información que Recopilamos

Cámara (android.permission.CAMERA)

  • Propósito: La aplicación solicita acceso a la cámara exclusivamente para permitirte capturar fotos de productos durante el registro en la base de datos.
  • Datos almacenados: Las imágenes capturadas se convierten a formato Base64 y se almacenan localmente en la base de datos Firebase Firestore de tu empresa.
  • Uso: Se utiliza únicamente para documentación de productos dentro de tu empresa y no se comparte con terceros externos.
  • Eliminación: Puedes eliminar las imágenes en cualquier momento eliminando el producto asociado.

Datos de Productos

  • Códigos de barras
  • Referencias de productos
  • Nombres de productos
  • Precios (base, recomendado, venta)
  • Categorías de productos
  • Fechas de creación de registros
  • Imágenes de productos (en formato Base64)

3. Almacenamiento de Datos

Todos los datos se almacenan en Firebase Firestore, un servicio de Google Cloud:
  • Los datos están encriptados en tránsito mediante HTTPS
  • Se aplican las políticas de seguridad automáticas de Firebase
  • Las imágenes se almacenan codificadas en Base64, no en servidores de almacenamiento separados
  • Acceso restringido solo a usuarios autorizados de tu empresa con credenciales válidas

4. Permisos Utilizados

Permiso Motivo del Uso Datos Procesados
android.permission.CAMERA Captura de fotos de productos Imágenes de productos (almacenadas localmente)
android.permission.INTERNET Conexión con Firebase para sincronización de datos Datos de productos, configuración
android.permission.ACCESS_NETWORK_STATE Verificar estado de conexión a internet Estado de conectividad

5. Seguridad de Datos

  • La aplicación utiliza autenticación mediante Firebase con reglas de seguridad en modo test
  • Los datos se almacenan en servidores seguros de Google Cloud con encriptación
  • Las conexiones de red utilizan protocolo HTTPS encriptado
  • No se comparte información con terceros sin consentimiento explícito de tu empresa
  • No se integran publicidades, análisis de terceros ni servicios de seguimiento

6. Datos Personales

Esta es una aplicación empresarial privada. Solo los usuarios autorizados por tu empresa pueden acceder a los datos. La aplicación no recopila datos personales (nombre, correo electrónico, teléfono) más allá de lo necesario para gestionar productos en la base de datos de tu empresa.

7. Retención de Datos

Los datos se retienen en Firestore mientras tu empresa mantenga la base de datos activa. Puedes:

  • Eliminar productos individuales dentro de la aplicación
  • Solicitar la eliminación completa de la base de datos a través de Google Cloud Console
  • Exportar datos en cualquier momento antes de la eliminación

8. Derechos del Usuario

Como usuario de una aplicación empresarial, tienes derecho a:

  • Acceder a tus datos en cualquier momento a través de la aplicación
  • Modificar o eliminar datos de productos
  • Solicitar información sobre cómo se almacenan tus datos
  • Contactar con tu administrador de empresa para consultas sobre privacidad

9. Cambios en la Política de Privacidad

Nos reservamos el derecho de actualizar esta política de privacidad en cualquier momento. Los cambios significativos se notificarán a los usuarios a través de:

  • Notificaciones dentro de la aplicación
  • Actualización de versión en Google Play Store
  • Comunicación directa a administradores de empresa

10. Contacto

Para preguntas sobre esta política de privacidad o prácticas de datos de la aplicación, contacta con tu administrador de empresa o equipo de TI.
